Automated Testing

Building a Modern CI infrastructure

Created by Thanh Ha / @zxiiro

## Agenda * OpenDaylight Infrastructure * Gerrit * Jenkins * Jenkins Job Builder * Vagrant * Scaling Jenkins
## OpenDaylight Infrastructure * The baseline for all Linux Foundation projects * Based on OpenStack's infrastructure * Infrastructure as an Open Source project
## Gerrit * Core to code review based workflow * +2 CR voting for project committers * +1 CR voting for all users * +1 Verified voting for CI System (Jenkins / Zuul) * Optionally launch jobs via Gerrit comments
## Jenkins automated builds and reporting * Dynamic slaves * Reproducible builds * JClouds plugin * Reports build status to Gerrit
## Jenkins job types * Verify * Merge * Integration * Periodic - Sonar - CLM - Daily
## Jenkins Job Builder * Manages jobs * Powerful templating system * YAML configuration stored in Gerrit * Code Review possible * Sandbox Jenkins for job testing
## Vagrant (Virtual Machine Management) * Stored in Gerrit * Code Reviewed * Reproducible VM configuration * Shared with users
## Scaling Jenkins * Jenkins Web UI is single threaded * SSH slaves take up 2 threads per slave * Zuul - Replaces Jenkins Gerrit Trigger plugin - Job Gating and Pipelines * Gearman * Nodepool - Replaces JClouds plugin - Slave management
